# Images Folder

## Logo File

Please add your logo image file here with the name: `logo.png`

The logo should be:
- Format: PNG (recommended) or other web-compatible format
- Recommended size: 32x32 pixels or larger (will be scaled down)
- The logo will be displayed in the sidebar next to "Lucrative" text

## File Location

Place your logo file as: `web/src/assets/images/logo.png`

The Sidebar component will automatically import and display this logo.
